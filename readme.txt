== Description ==

"Piwik tracking, by Mente Binaria" is a simple plugin whose only purpose is to add the Piwik tracking code to your website.

This will not install Piwik, so you will need to do that yourself. For more information about Piwik, visit [piwik.org](http://piwik.org/).

== Installation ==

This section describes how to install the plugin and get it working.

1. Unzip "piwik-tracking-by-mb.zip" and upload the "piwik-tracking-by-mb" folder to your plugin directory (usually "/wp-content/plugins/").
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Visit the "Piwik tracking" page in the "Settings" section and adjust the parameters. Don't forget to click "Save Changes" when you're done.
